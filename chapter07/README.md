# 7. 성능 튜닝

리엑트는 기본적으로 성능을 고려하여 설계됨. DOM 조작 횟수 최소화 등.

## 조정 프로세스
리엑트는 컴포넌트의 상태가 변경될 때마다 렌더링 프로세스가 트리거되어 동작한다. 이 때 가상 DOM이 생성 되고 기존의 가상 DOM과 비교 연산을 통해 변경이 필요한 DOM요소를 식별한다. 이러한 과정을 조정(reconcilation) 이라고 한다.
